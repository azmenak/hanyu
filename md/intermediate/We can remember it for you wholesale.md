# We can remember it for you wholesale
**Intermediate** level
### Chinese Text
第一部
甲：Copenhagen，放开那个女孩！
乙：什么女孩？
甲：她在那儿，不，这儿。
乙：这儿没有女孩，这只是你的大脑错乱。
甲：我的头，我的头，我的头疼死了。
乙：你应该回家休息。
甲：你对我的脑子做了什么？而且我告诉过你，放开那个女孩！
第二部
甲：帅哥，救救我！
乙：Copenhagen，放开那个女孩。
丙：她不是女孩。还是，你可能不记得了....
乙：什么？
丙：我们移除你记忆的同时，也剥夺了你辨认性别的能力。
乙：我怎么啦？
丙：很遗憾地告诉你，你的老婆其实是个男人，而且她为我工作。
