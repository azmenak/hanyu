# The Argument
**Intermediate** level
### Chinese Text
女：咳，老板走了没?
男：终于走了。
女：哎呀，太好了。走。咱们吃饭去!
男：你还没工作完呢?
女：哎呀，老板都走了谁还要工作呢? 走吧, 走吧, 别假正经了!
男：好，走吧。
