# The Spanish Main
**Intermediate** level
### Chinese Text
船长：啊，终于到了要挖宝藏的时候了！
大副：船长，我们马上就要变成富翁了！
船长：对！快，拿铲子来！
大副：什么铲子？
船长：挖宝藏的铲子呀！
大副：不是去年冬天已经烧了吗？
船长：什么？你是说你没买新的吗？
大副：可是我们哪儿有钱买新的？钱都埋在地底下呢！
