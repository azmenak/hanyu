# The Public Security Alarm
## *absolute-beginners* level

### Chinese Text
［ 第一部 ］
男人：二十一块钱。
女人：我也需要一个袋儿。
男人：好，没问题。
女人：不用客气，我自己来。
［ 第二部 ］
女人：一共是二十一块钱。
男人：可以给我拿个袋儿吗？
女人：好。
男人：不用不用，我自己装。
