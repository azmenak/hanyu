# It Happened One Night
**Elementary** level
### Chinese Text
甲：嘿，你知道昨晚谁赢了奥斯卡吗？
乙：我听说《盗梦空间》是大赢家。
甲：没错。
乙：它赢了最佳影片奖。
甲：对。
乙：最佳导演奖。
甲：应该的。
乙：最佳表演奖。
甲：是。
乙：还有最佳外语影片奖。
甲：外语影片奖？
乙：对，全都赢了。
